Qualifier: tmp

tmp can be set on variables to keep them from persisting in the vars.json. This can be helpful for any variables that may grow to a large size. Note that renew can also be part of a strategy for handling potentially large variables.

Currently, tmp cannot be applied to the variables that are created by functions behind the scenes for their own purposes. These variables can be cleared using tmp but only if done more intentionally by declaring the variable yourself. Those behind the scenes variables that are named by an random ID cannot be cleared by tmp at this time.

Availability

Functions Headers Variables
No No Yes

Examples

This csvpath sets the @soda variable to "Coke" on odd lines and clears @soda at the end of the run so it won't be stored in vars.json.

$[1*] [ odd() -> @soda.tmp = "Coke" ]
