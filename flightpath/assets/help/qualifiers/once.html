Qualifier: once

once permits a match component to evaluate one and only one time.

After a component that carries the once qualifier has its only evaluation it has no future contribution to line matching. The effect is as if the once qualifier is replaced thereafter with the nocontrib qualifier.

Availability

Functions Headers Variables
Yes No No

Examples

A side-effect like print() might use once to register a situation one time, rather than adding noise by continuing to announce the situation.

This csvpath prints an alert if a header value goes above a certain level; however, it only prints it one time.

$[*][ gt(#temperature, 100) -> print.once("Too hot!") ]
