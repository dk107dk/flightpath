Qualifier: renew

renew resets a latched match component.

Availability

Functions Headers Variables
No No Yes

Examples

This csvpath only says hi to Eddie. A the end of a run, if the last line's first name is not Eddie @m will hold the first name and @x will be None.

As a simple analogy, CsvPath Language variables are like HTTP session variables, in that they last across all lines in the whole document. However, the renew qualifier makes a variable act like a request-scoped variable, resetting after every line.

$[*][ @m = #"First Name" @m == "Eddie" -> @x.renew = yes() @x.asbool -> print("Hi Eddie") ]
