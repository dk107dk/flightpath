Qualifier: renew

renew resets a variable match component at the start of each line. You can add the renew qualifier to any one use of the variable and have the same effect of clearing the variable before the line starts.

Availability

Functions Headers Variables
No No Yes

Examples

This csvpath only says hi to Eddie. A the end of a run, if the last line's first name is not Eddie @m will hold the first name and @x will be None.

As a simple analogy, CsvPath Language variables are like HTTP session variables, in that they last for the duration of a run, across all lines in the whole document. However, the renew qualifier makes a variable act like a request-scoped variable, resetting for every line.

$[*][ @m = #"First Name" @m == "Eddie" -> @x.renew = yes() @x.asbool -> print("Hi Eddie") ]
