Qualifier: nocontrib

nocontrib prevents a match component from contributing to the match decision for each line seen.

nocontrib enables conditional side-effects. In other words, you can add nocontrib to the left-hand side of a do-when expression in order to make the side-effect happen without the line matching.

Availability

Functions Headers Variables
Yes Yes Yes

Examples

Say we have a data with 10 lines and one of the 9 data lines has georgia as the value of the firstname header. This csvpath says hello only to Georgia, but it matches all the lines, not only Georgia. $[1*][ #firstname.nocontrib == "georgia" -> print("Hi Georgia!") ]
