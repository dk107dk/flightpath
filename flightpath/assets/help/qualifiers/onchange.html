Qualifier: onchange

onchange allows match components to activate when their value changes from one line to the next.

Availability

Functions Headers Variables
Yes No Yes

Examples

Using onchange a print statement can be activated only when it would print something different.

$[*][ ~ this statement prints only the first city ~ print.onchange.once("The city is $.headers.city") ~ this statement prints only when the city changes ~ print.onchange("The city is $.headers.city") ~ this statement prints only when the city changes and the line matches ~ print.onchange.onmatch("The city is $.headers.city") ]

onchange makes a variable assignment contribute to matching. In this case a line only matches when the name is Eddie or Aimee or switches to or from one of those names.

$[*][ @oc.onchange = in(#"First Name", "Eddie|Aimee") print.onmatch("oc: $.variables.oc, count: $.csvpath.count_matches") ]
