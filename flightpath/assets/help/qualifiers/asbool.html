Qualifier: asbool

The asbool qualifier makes a match component's output be considered a boolean value.

Availability

Functions Headers Variables
Some, see specific function docs. Yes Yes

Examples

Variables used alone always match if they exist. Using asbool we can check that the value of a variable is true or false. $[*][ @a = 0 @a.asbool -> print("0 becomes False, even though @a exists.") ]
Say we have a file with four names and one sally. With asbool we will say hi to Sally once. Without asbool we will say hi four times $[*][ @in.asbool = in(#firstname, "sally") print.onmatch("Hi Sally") ]
