Qualifier: latch

latch prevents a variable's value from changing once it has been set. Latch is a guard, not a validation. That means it prevents a variable from changing, but doesn't cause an error if the value you attempt to assign would have changed the variable's value.

Availability

Functions Headers Variables
No No Yes

Examples

The value of a remains 0. The value of b counts up as the number of lines seen increases. $[*][ @a.latch = line_number() @b = line_number() ]
