Qualifier: notnone

notnone disallows a None value for a match component. None is the value of the none() function and is roughly equivalent to the Python None.

Within a line() structural schema notnone plays the same role as DDL's NOT NULL designation.

Availability

Functions Headers Variables
Yes No Yes

Examples

A line() schema can use notnone to require a value.

This schema requires the first header to be called firstname, have the string(), and always have a value.

$[*][ line( string.notnone(#firstname), wildcard("*") ) ]

Assignments can be conditional to a value being present.

Here the @firstname only changes if there is a new non-None value in the #firstname header.

$[*][ @firstname.notnone = #firstname ]
