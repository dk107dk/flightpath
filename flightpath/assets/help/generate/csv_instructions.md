## AI Instructions

This dialog generates CSV data based on one or more CsvPath Language statements. The data is completely made up and matches your validation rules.

Use this text box to provide more specific instructions to AI. You can use this capability to fine tune your test set to a specific testing purpose. For example, you could generate data for a list of US addresses, providing the instruction to only include addresses from New England states. Without providing that more specific instruction your test data would be generated with a random set of states from across the country.
