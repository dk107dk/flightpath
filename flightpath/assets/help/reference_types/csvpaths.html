Reference type: csvpaths

The csvpaths reference type points to named-paths groups. It is used to create run references.

The main benefit of using csvpaths type references is that you can choose what csvpaths within the group are used in the run. Individual csvpaths can be selected, or you can select a starting or an ending csvpath.

Usage

This reference starts the run at the 3rd csvpath. $shippment-tracking.cvspaths.2:from
This reference starts the run at the 0th csvpath and stops it on the csvpath identified as readings. $wind-troposphere.cvspaths.readings:to
