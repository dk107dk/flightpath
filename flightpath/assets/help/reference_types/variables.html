Reference type: variables

The variables reference type accesses values in the currently executing csvpath's set of variables.

Csvpath variables are set within CsvPath Language using the @ sign. A variable named @city holds user defined data that presumably relates to cities. Once set, variables typically hold values, possibly changing, for the duration of the run.

A reference to the variables namespace is useful in at least two ways:

  • Variable references can be used in print() statements
  • A variable reference can pull data in from the variables of a past run
Currently, references to variables from past runs only accesses the most recent run. In the future the capability to access any past run will probably be added.

Usage

This reference prints the @description variable. $[*][ @description = #model #color -> @description = concat(#color, " ", #model) print("The car is a $.variables.description") ]
