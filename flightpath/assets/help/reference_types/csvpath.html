Reference type: csvpath

The csvpath reference type points to runtime data for the currently running csvpath. It is used exclusively in the print() function.

See the runtime print fields help topic for more about what data is available.

Usage

This prints the line number. $[*][ odd() -> print("This is an odd numbered row: $.csvpath.line_number") ]
This prints the validity status at the end of a run. $[*][ last.nocontrib() -> print("This file is valid: $.csvpath.valid") ]
