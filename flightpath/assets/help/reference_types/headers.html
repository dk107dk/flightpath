Reference type: headers

The headers reference type points to the header values for the current line of data in the currently running csvpath. They are used exclusively in the print() function.

Header names can have spaces and therefore may need to be quoted. Within print() statements header names can be quoted only with single quotes.

Usage

This reference prints the current header value. print("This is the first name: $.headers.first_name") ]
This csvpath prints a header that has a space in its name. $[*][ last.nocontrib() -> print("The value of first name is: $.headers.'first name' ") ]
