Print field: lines_collected

lines_collected is the number of lines that were collected up to the point the print function is called. Collected lines are returned in the next() and collect() methods (and derivatives). They are the lines in the data.csv file that is created when you run a named-paths group.

Typically this number will be the same as match_count, but if you are collecting unmatched lines it will differ. Collecting unmatched lines is done using the unmatched-mode directive in an external comment.

Usage

Use this field like this. $[0][ print("$.csvpath.lines_collected") ]
