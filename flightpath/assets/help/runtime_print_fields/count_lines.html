Print field: count_lines

count_lines is the number of lines considered data by the CsvPath Framework that have been evaluated up to the moment the print function executes. In the CsvPath Framework a "count" is generally 1-based, in contrast with a field like line_number which is 0-based. 0-based is keeping with how you would programmatically address a line. 1-based matches how people count.

Usage

Use this field like this. $[0][ print("$.csvpath.count_lines") ]
