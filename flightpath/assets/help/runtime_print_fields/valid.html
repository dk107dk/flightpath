Print field: valid

valid prints true if the csvpath is considered valid, according to the csvpath writer. For valid to return false, the csvpath writer would have to have:
• Used the fail() function
• Set errors to result in failure in config/config.ini -and- for there to have been an error
• Set the validation-mode to include fail -and- for there to have been an error
If none of those things happened, the data file is considered valid.

Usage

Use this field like this. $[0][ print("$.csvpath.valid") ]
