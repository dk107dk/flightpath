Print field: quotechar

quotechar is the CSV quotechar used to wrap values that may have breaking characters. Typically it is the double-quotes character. Quotechar is not used with Excel and other non-CSV formats. The quotechar will never change during a run.

Usage

Use this field like this. $[0][ print("$.csvpath.quotechar") ]
