Print field: identity

identity is the ID of the csvpath being executed. It is set as an index automatically. You can use a word or words by setting a metadata field in an external comment. The metadata field name must be one of ID, Id, id, NAME, Name, or name.

The identity of the csvpath is shown in error and validation messages and in various metadata.

Usage

Use this field like this. ~ id: my validation ~ $[0][ print("$.csvpath.identity") ]
