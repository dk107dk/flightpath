Symbol: "

Quotes are used to create string tokens within the match part of a csvpath. A string can stand alone, be assigned to a variable, used in a function, or compared.

Quotes are also used with header names that have spaces in them. A single or double quote can wrap a header name.

Usage

This is a string assigned to a variable. The variable is used in a print statement's string. $[*][ @hi = "hello" print("$.variable.hi friend") ]
This header name has a space so it must be wrapped in single or double quotes. $[*][ #"my name" == "Fred" -> print("hello Fred") ]
Remember that within a print string you must wrap header names with spaces in single-quotes, not double-quotes. $[*][ print("Hello, $.headers.'my name'!") ]
