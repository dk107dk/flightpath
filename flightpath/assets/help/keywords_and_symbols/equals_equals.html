Symbol: ==

The equality test. == can be used to test a line for match or as an input to another match component.

Usage

Equals can determine if a line matches. This example does not match any lines. @a = "day" @a == "night"
In assignments you must use the equals() or eq() function, not ==. Rather than: @a = "day" = "night" do: @a = eq("day","night") In all other cases use ==.
