Symbol: ->

-> is the when-do symbol. When-do is essentially CsvPath Language's if statement. When the left-hand side of when-do evaluates to True, the right-hand side happens.

Usage

When a name matches, print a message. $[*][ #name == "Fred" -> print("Hi, Fred") ]
When the last line is seen, print a summary.

Note the nocontrib qualifier that keeps last() from contributing to lines matching or not matching.

If we didn't use nocontrib we would only count the last line, and only then if the name was Fred.

$[*][ #name== "Fred" -> counter.tick() last.nocontrib() -> print("Saw $.variables.tick Freds") ]
