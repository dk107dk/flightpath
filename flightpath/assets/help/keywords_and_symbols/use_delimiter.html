Keyword: use-delimiter

use-delimiter is a CsvPath Framework metadata field. It configures an individual csvpath for a file to have a certain delimiter.

To use use-delimiter create a metadata comment above your csvpath. In it, add use-delimiter:delimiter-word, where delimiter-word is one of the words in the left-hand column, mapping the character on the right:
bar |
pipe |
semi-colon ;
semicolon ;
comma ,
tab the tab character
space the space character

This metadata instruction does not give all the options that are available programmatically; however, these common delimiters will satisfy the large majority of cases.

Usage

This csvpath will always run within FlightPath using the identified file when you right-click and select Run or hit ctrl-r. ~ use-delimiter: pipe ~ $[0][ yes() ]
