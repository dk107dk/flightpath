Keyword: test-delimiter

test-delimiter is a metadata field specifically for FlightPath. It tells FlightPath that the file or files you will be running the current csvpath on have a certain delimiter.

To use test-delimiter create a metadata comment above your csvpath. In it, add the field with test-delimiter:delimiter-word, where delimiter-word is one of the words in the left-hand column, mapping the character on the right:
bar |
pipe |
semi-colon ;
semicolon ;
comma ,
hash #
percent %
star *
asterisk *
at @
tilde ~
tab the tab character
int any positive char code integer

The int code allows you to use any Unicode character. In most cases, pipe, semi-colon, and comma are enough. Comma is the default. There is no need to specify that the delimiter to use is a comma.

Usage

This csvpath will always run within FlightPath using the identified file when you right-click and select Run or hit ctrl-r. ~ test-delimiter: pipe ~ $[0][ yes() ]
