Keyword: test-quotechar

test-quotechar is a metadata field specifically for FlightPlan. It tells FlightPlan that the file or files you will be running the current csvpath on have a certain quotechar.

To use test-quotechar create a metadata comment above your csvpath. In it, add the field with test-quotechar:quotechar-word, where quotechar-word is one of the words in the left-hand column, mapping the character on the right:
quotes "
quote "
single-quotes '
singlequotes '
single-quote '
singlequote '
tick `
int any positive char code integer

The int code allows you to use any Unicode character. In most cases, single or double quotes are enough. Quotes is the default. There is no need to specify that the quotechar to use is a quote character.

Usage

This csvpath will always run within FlightPlan using the identified file when you right-click and select Run or hit ctrl-r. ~ test-quotechar: singlequotes $[0][ yes() ]
