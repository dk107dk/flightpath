Keyword: PRINTOUT

The print function can send printouts to any number of printers. (A Printer is a CsvPath Framework class that collects lines of output). This capability is mainly used in named-paths group runs, not one-off validations.

Every run that has any output has a printouts.txt file. The printout.txt file contains all the printed strings separated in blocks for each printer. The blocks are separated by single line in the form: ---- PRINTOUT: name. Where name is the name used for the printer.

The print() and error() functions typically print to the default printer. You can pass a specific printer name to print() as a second argument.

Usage

This csvpath prints to three print streams. $[0][ print("This goes to default") error("This also goes to default, as well as throwing an error into errors.json") print("This goes to micky-mouse operation", "micky-mouse operation") print("And this message goes to popsicle stand", "popsicle stand") ]
This is the printouts.txt resulting from the csvpath above. ---- PRINTOUT: default This goes to default This also goes to default, as well as throwing an error into errors.json ---- PRINTOUT: micky-mouse operation This goes to micky-mouse operation ---- PRINTOUT: popsicle stand And this message goes to popsicle stand
