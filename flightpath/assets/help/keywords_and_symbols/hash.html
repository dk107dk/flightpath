Symbol: #

The hash mark is used in two ways:

  • As the first character of a header name to access the header value for a line
  • In match component references following a named-results name to indicate a specific csvpath

Usage

This shows setting a variable with the given_name header's value for a line. $[*][ @hi = #given_name ]
This reference pulls a variable called who from the csvpath identified as people in the named-paths group my_named_paths. $[*][ push("names", $my_named_paths#people.variables.who) ]
