Keyword: use-quotechar

use-quotechar tells CsvPath Framework that the current csvpath uses a certain quotechar. This is a CsvPath Framework feature, rather than a FlightPath-specific feature.

To use use-quotechar create a metadata comment above your csvpath. In it, add the field with use-quotechar:quotechar-word, where quotechar-word is one of the words in the left-hand column, mapping the character on the right:
quotes "
quote "
single-quote '
singlequote '
tick `

This metadata instruction does not give all the options that are available programmatically; however, these common quote chars will satisfy the large majority of cases.

Usage

This csvpath will always run within FlightPath using the identified file when you right-click and select Run or hit ctrl-r. ~ use-quotechar: singlequotes $[0][ yes() ]
