Symbol: '

Single quotes are used to create are used with header names that have spaces in them in two ways:

  • In general, single (or double) quotes can wrap a header name with spaces
  • Within print() only single quotes can wrap header names with spaces

Usage

This header name has a space so it must be wrapped in single or double quotes. $[*][ #'my name' == "Fred" -> print("hello Fred") ]
Within a print string you must wrap header names with spaces in single-quotes, not double-quotes. $[*][ print("Hello, $.headers.'my name'!") ]
