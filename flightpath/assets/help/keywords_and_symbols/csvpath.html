Keyword: CSVPATH

When you define multiple csvpaths in a single file they must be separated with: ---- CSVPATH ----. A preceding and/or trailing ---- CSVPATH ---- is permitted but not required.

Csvpaths are loaded into CsvPath Framework as a named-paths group in one of several ways:

  • One by one
  • As the set of csvpath files in a directory
  • As a JSON dict mapping named-paths group names to csvpath files
  • As multiple cvspaths contained in a single file
In each case, the csvpaths of the named-paths group are added to a single group.csvpath file. Within that file the csvpaths are separated by ---- CSVPATH ----.

The cvspaths in the same file are run together as a named-paths group against one data file.

Usage

This is three csvpaths within one file $[*][ @hi = #given_name ] ---- CSVPATH ---- $[*][ #sign == "scorpio" -> stop_all() ] ---- CSVPATH ---- $[*][ print("This is your lucky day", "horiscope")
