Mode: source-mode

Source mode determines where the data for a csvpath comes from. It is used only in running named-paths groups.

There are two source options:

  • The original named-file data fed into the named-paths group (this is source-mode's default)
  • The output from the csvpath preceding the csvpath where source-mode is set

Keep in mind that source-mode: preceding is one of two ways to pass data from one csvpath to the next. The other way is to do a breadth-first run. A breadth-first run is one of the _by_line methods on the CsvPaths class: collect_by_line(), fast_forward_by_line(), and next_by_line(). The breadth-first methods pass a line's data to all csvpaths in a named-paths group before moving on to the next line. Each csvpath can modify the data in the line before passing it to the next csvpath. Both serial and breadth-first runs are supported in FlightPath.

Usage

This named-paths group passes data from one to the next. The result is 1/2 of 1/2 of the starting data.

~ id: just the odds ~ $[*][ odd(line_number()) ] ---- CSVPATH ---- ~ source-mode:preceding id: quartering the set ~ $[*][ odd(line_number()) ]
