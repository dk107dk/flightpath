Mode: run-mode

run-mode blocks a csvpath from running. The value is principally during testing when you may not want a certain csvpath to run with the other csvpaths in a group.

Usage

The not-going-to-run csvpath will not participate in its named-paths group. let's run! will participate, as normal.

The run-mode default is run so you wouldn't typically see that setting.

~ id: not-going-to-run run-mode:no-run ~ $[*][ print("I'm going to sit this one out") ] ---- CSVPATH ---- ~ id: let's run! ~ $[*][ print("I'm ready to go") ]
